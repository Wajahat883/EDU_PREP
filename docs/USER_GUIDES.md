## Student Guide

### Getting Started

#### Creating Your Account

1. Visit [eduprep.com](https://eduprep.com)
2. Click **Sign Up**
3. Enter your details:
   - Full name
   - Email address
   - Strong password (min 8 characters, 1 uppercase, 1 number)
   - Grade level
4. Click **Create Account**
5. Verify your email by clicking the link sent to your inbox

#### First Login

1. Enter your email and password
2. Complete your profile:
   - Select your subjects of interest
   - Choose your learning goals
   - Indicate your availability (for scheduling)
3. Click **Get Started**

### Dashboard Overview

**Main Dashboard** shows:

- **Progress Overview**: Your overall learning progress by subject
- **Recommended Questions**: Personalized questions based on your performance
- **Active Learning Path**: Current adaptive learning path
- **Recent Tests**: Your 5 most recent test attempts
- **Streak Counter**: Consecutive days of practice

### Taking Practice Questions

#### Find Questions

1. Click **Practice** in main menu
2. Filter by:
   - **Subject**: Mathematics, English, Science, etc.
   - **Difficulty**: Easy, Medium, Hard
   - **Topic**: Specific topic within subject
3. Click **Start** on desired question

#### Answer Question

1. Read question carefully
2. Review all options
3. Select your answer
4. Click **Submit**
5. View:
   - Correct/incorrect indicator
   - Explanation
   - Time spent
   - Related questions (if missed)

#### Review Performance

- **Question History**: See all questions attempted
- **Accuracy by Topic**: View performance breakdown
- **Time Analysis**: See average time per question
- **Weak Areas**: Automatically identified struggling topics

### Taking Tests

#### Browse Available Tests

1. Click **Tests** in main menu
2. View:
   - Test name
   - Number of questions
   - Duration
   - Passing score
   - Your previous attempts (if any)

#### Start a Test

1. Click on test
2. Review:
   - Instructions
   - Number of questions
   - Time limit
   - Passing score
3. Click **Start Test**
4. Timer begins (appears top right)

#### During Test

- Questions appear one at a time
- Select answer and click **Next**
- Mark questions for review
- View time remaining
- Cannot go back to previous questions
- Test auto-submits when time expires

#### Review Results

After test completion, view:

- **Score**: Your percentage correct
- **Pass/Fail**: Whether you met passing score
- **Question-by-Question Review**: See correct answers
- **Performance Insights**:
  - Topics you excelled in
  - Topics needing improvement
  - Time management feedback

### AI-Powered Features

#### Personalized Recommendations

**Access**: Dashboard → **Recommended Questions**

System recommends questions based on:

- Your performance history
- Difficulty level appropriate for you
- Topics you need to strengthen
- Questions similar students completed
- Spaced repetition principles

**Tips**:

- Start with recommended difficulty
- Complete 5-10 before progressing
- Review explanations thoroughly

#### Adaptive Learning Paths

**Access**: Dashboard → **Learning Path** or **AI** → **Create Path**

**Starting a Path**:

1. Click **Create New Path**
2. Select subject (Math, English, Science, History)
3. Choose target level:
   - Beginner (first time)
   - Intermediate (some experience)
   - Advanced (mastery level)
4. System generates personalized path

**Following Your Path**:

- Complete modules in order
- Each module has mastery requirement (75%)
- Difficulty adjusts automatically:
  - Increases if scoring > 85%
  - Decreases if scoring < 50%
- Milestones mark major progress points
- Estimated completion date shown

#### Performance Predictions

**Access**: Test page → **AI Prediction**

Before taking a test, system predicts:

- **Predicted Score**: Your likely score (with confidence%)
- **Score Range**: Likely score ± margin of error
- **Success Probability**: Likelihood of passing
- **Topics to Review**: Weak areas to study before test
- **Prep Time Needed**: Hours of practice recommended

#### Study Schedule Recommendations

**Access**: Test page → **Study Schedule**

Get personalized prep plan:

- **Days Until Test**: How many days to prepare
- **Daily Study Schedule**: Topics to study each day
- **Estimated Time**: Hours needed daily
- **Best Study Times**: When you perform best
- **Study Tips**: Evidence-based recommendations

### Analytics & Progress

#### View Statistics

1. Click **Analytics** in main menu
2. View:
   - **Overall Stats**: Questions attempted, accuracy, time spent
   - **By Subject**: Performance in each subject
   - **By Topic**: Deep dive into specific areas
   - **Trends**: 7-day, 30-day, all-time trends

#### Interpret Charts

- **Accuracy Trend**: Should generally go up
- **Time Trend**: Average time per question
- **Volume**: How many questions you're completing
- **Subject Breakdown**: Where you're strongest/weakest

### Subscription & Payment

#### View Subscription

1. Click **Account** → **Subscription**
2. See:
   - Current plan
   - Renewal date
   - Features available

#### Upgrade Plan

1. Go to **Account** → **Subscription**
2. Click **Upgrade**
3. Choose new plan:
   - **Free**: Limited access
   - **Basic**: $9.99/month - 5 tests, 100 questions
   - **Premium**: $19.99/month - Unlimited + AI coaching
4. Add payment method
5. Click **Subscribe**

## Instructor Guide

### Setting Up Your Course

#### Create a Course

1. Go to **Instructor Dashboard**
2. Click **Create Course**
3. Enter:
   - Course name
   - Subject
   - Description
   - Course code (for students to join)
   - Enrollment limit
   - Duration
4. Click **Create**

#### Add Students

**Option 1: Share Course Code**

- Students enter code during signup
- Auto-added to your course

**Option 2: Manual Invite**

1. Go to Course → **Students**
2. Click **Invite Students**
3. Enter email addresses (comma-separated)
4. Select role: Student, Teaching Assistant
5. Click **Send Invites**

**Option 3: Bulk Upload**

1. Click **Upload CSV**
2. Format: `email,firstName,lastName,role`
3. Upload file
4. Review and confirm

#### Create Custom Questions

1. Go to **Content** → **Create Question**
2. Enter:
   - Question text
   - Subject and topic
   - Difficulty level
   - 3-4 options (mark correct answer)
   - Explanation (shown after answering)
   - Time estimate (minutes)
   - Tags (for organization)
3. Click **Save as Draft**
4. Preview or **Submit for Approval**

**Tips for Good Questions**:

- Clear, unambiguous wording
- Realistic difficulty progression
- Comprehensive explanations
- Varied option types (avoid all numeric, etc.)
- Appropriate length

#### Create Tests

1. Go to **Assessments** → **Create Test**
2. Configure:
   - Test name and description
   - Duration (minutes)
   - Total questions
   - Passing score
   - Difficulty mix
3. Add questions:
   - From question bank
   - Search and select
   - Add custom questions
   - Reorder by dragging
4. Set availability:
   - Available immediately or on schedule
   - End date/time
   - Allow review after completion
   - Show answers
5. Click **Publish**

### Managing Your Class

#### Assign Tests

1. Go to Course → **Assignments**
2. Click **Assign Test**
3. Select test
4. Choose students/groups
5. Set deadline
6. Click **Assign**

#### Monitor Progress

**Class Dashboard** shows:

- Average class accuracy
- Student completion rates
- Topics class is struggling with
- Class trends over time

**Individual Student View**:

1. Click student name
2. See:
   - All questions completed
   - All tests taken
   - Performance trends
   - Strong/weak topics
   - Time spent

#### Grade Tests

1. Go to **Assignments** → Select test
2. See list of submissions
3. Click student name to:
   - Review answers
   - View time spent per question
   - Add feedback comments
   - Adjust score if needed (for essay questions)
4. Click **Save**

#### Provide Feedback

1. During grading, click **Add Feedback**
2. Write constructive comments
3. Attach resources (videos, documents)
4. Student receives notification

### Advanced Features

#### Plagiarism Detection

When students submit essays:

1. System automatically checks for plagiarism
2. View **Plagiarism Report**:
   - Similarity percentage
   - Risk level (Low/Medium/High/Critical)
   - Flagged sections
   - Sources matched
3. Click on flagged section to see source
4. Make academic integrity decision

#### Performance Analytics

1. Go to **Analytics**
2. View:
   - **Class Overview**: Overall performance metrics
   - **Subject Analysis**: Performance by subject
   - **Topic Mastery**: Topics class has mastered/struggling
   - **Test Performance**: Results by test
   - **Trends**: Class improvement over time

**Export Data**:

1. Click **Export**
2. Choose format (CSV, Excel, PDF)
3. Select date range
4. Download file

#### Create Study Groups

1. Go to **Groups** → **Create Group**
2. Enter:
   - Group name
   - Members
   - Focus subject/topic
3. Click **Create**

Groups allow:

- Private messaging
- Shared resources
- Collaborative studying
- Group performance tracking

## Administrator Guide

### User Management

#### Add Users

**Single User**:

1. Go to **Users** → **Add User**
2. Enter email, name, role
3. System sends invitation
4. User completes signup

**Bulk Import**:

1. Go to **Users** → **Bulk Upload**
2. Download template CSV
3. Fill in user details
4. Upload file
5. Review conflicts
6. Click **Import**

#### Manage Users

1. Go to **Users** → Select user
2. Options:
   - **Edit**: Change name, email, role
   - **Reset Password**: Send reset link
   - **Disable**: Deactivate account
   - **View Activity**: See login history, usage
   - **Export Data**: Download user data (GDPR)

#### User Roles

- **Student**: Can take tests, practice
- **Instructor**: Can create content, grade
- **Admin**: Full platform access
- **TA (Teaching Assistant)**: Can help with grading

### Content Management

#### Question Bank

1. Go to **Content** → **Questions**
2. View all questions with:
   - Subject, topic, difficulty
   - Creator, status (draft/approved)
   - Usage count
   - Ratings

**Actions**:

- **Search**: Find questions by text
- **Filter**: By subject, difficulty, status
- **Approve**: Verify and approve new questions
- **Archive**: Hide old questions
- **Duplicate**: Create similar question

#### Question Quality

Review and approve questions based on:

- ✓ Clear, unambiguous wording
- ✓ Correct answer accuracy
- ✓ Helpful explanation
- ✓ Appropriate difficulty
- ✓ Proper formatting

#### Test Management

1. Go to **Content** → **Tests**
2. View all tests with:
   - Number of questions
   - Duration
   - Average score
   - Times taken
   - Passing rate

**Actions**:

- **View**: See full test
- **Edit**: Modify (only if not yet taken)
- **Archive**: Remove from circulation
- **View Results**: Analytics for all test attempts

### Analytics & Reporting

#### Platform Analytics

1. Go to **Analytics** → **Platform**
2. View:
   - **Users**: Total, active, by role
   - **Activity**: Daily questions, tests, logins
   - **Performance**: Average accuracy, passing rates
   - **Usage**: Popular subjects, peak times
   - **Revenue**: Subscription stats, MRR

#### Generate Reports

1. Go to **Reports**
2. Choose report type:
   - **User Activity**: Login, usage, performance
   - **Content Usage**: Questions, tests, popularity
   - **Performance**: Accuracy, progress, trends
   - **Financial**: Revenue, subscriptions, billing
3. Select date range
4. Click **Generate**
5. Download as PDF/CSV

#### Export Data

1. Go to **Data** → **Export**
2. Choose data type:
   - Users
   - Questions
   - Tests
   - Results
   - Analytics
3. Select format (CSV, JSON)
4. Download

### System Configuration

#### Settings

1. Go to **Settings** → **General**
2. Configure:
   - Platform name, logo, colors
   - Email domain
   - Time zone
   - Language

#### Email Configuration

1. Go to **Settings** → **Email**
2. Set:
   - Sender email
   - Sender name
   - Email templates
3. Send test email

#### Payment Settings

1. Go to **Settings** → **Payment**
2. Configure:
   - Stripe account
   - Plans and pricing
   - Tax settings
   - Billing emails

#### Security

1. Go to **Settings** → **Security**
2. Configure:
   - Password requirements
   - Session timeout
   - IP whitelist
   - 2FA requirement (roles)
   - Data retention policy

### Monitoring & Maintenance

#### System Health

1. Go to **System** → **Health**
2. View:
   - Service status (all green = healthy)
   - Database status
   - API response times
   - Error rates
   - Cache hit ratio

#### Logs

1. Go to **System** → **Logs**
2. View:
   - Activity logs
   - Error logs
   - Security logs
3. Filter by:
   - Date range
   - User
   - Action type
   - Status

#### Backups

1. Go to **System** → **Backups**
2. View scheduled backups
3. Manual backup:
   - Click **Backup Now**
   - Confirm
4. Restore backup:
   - Select backup
   - Click **Restore**
   - Confirm (data loss warning)

### Troubleshooting

#### Common Issues

**User locked out**:

- Click **Reset Password**
- User receives reset link
- Can create new password

**Questions not showing**:

- Check question status (must be approved)
- Check publishing settings
- Verify user permissions

**Tests not available**:

- Check test availability dates
- Verify user enrollment in course
- Check test permissions

**Payment issues**:

- Go to **Payments** → **Transactions**
- Click failed transaction
- See error reason
- Manual retry option available

#### Support

1. Go to **Help** → **Support**
2. Search knowledge base
3. Or submit support ticket
4. Contact support@eduprep.com for urgent issues
